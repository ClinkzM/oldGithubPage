PhantomJS - 可编程的无界面浏览器（官网在此）

重要通知: 除非另行通知，PhantomJS 项目已经搁置。 (详情)

PhantomJS 是一个可用 JavaScript 来编译的无界面网络浏览器。能在 Windows、macOS、Linux 和 FreeBSD 系统上运行。

使用 QtWebKit 渲染引擎（译注：QtWebKit 是一个用 Qt 封装了 Webkit 浏览器引擎的工具包），为各种不同的网络标准提供快速本地化的支持，比如处理 DOM 树、CSS选择器、JSON、Canvas 以及 SVG。

在 PhantomJS 通过以下一段简单的代码加载谷歌的主页，稍等一会，就能将主页生成图片。

            var page = require('webpage').create();
            page.open('http://www.google.com', function() {
                setTimeout(function() {
                    page.render('google.png');
                    phantom.exit();
                }, 200);
            });

PhantomJS 是以下的最佳实践：

页面自动化
使用标准DOM API或者常见库，比如jQuery，来访问网页并提取信息。
屏幕获取
程序抓取网络内容，包括 SVG 和 Canvas。带缩略图预览的网页页面截图。
无界面网页测试
使用框架进行功能测试，比如 Jasmine、QUnit、Mocha、WebDriver等等.
网络资源监控
监控页面加载和导出的标准HAR文件。使用YSlow和Jenkins自动执行性能分析。
